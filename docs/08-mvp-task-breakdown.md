# Weapon Gouelite MVP 任务拆分

## 里程碑 1：规格与工程基础

- 完成 MVP GDD、技术设计和任务拆分文档
- 初始化 Git 仓库与远端
- 初始化 Godot 项目与测试运行器

## 里程碑 2：核心数据与菜单

- 建立数值表与武器目录
- 建立存档、广告、当前局状态服务
- 完成主菜单与局外成长

## 里程碑 3：战斗闭环

- 玩家左右移动
- 自动攻击
- 敌人刷新与基础 AI
- 经验、升级与三选一
- 层数推进

## 里程碑 4：武器与 Boss

- 回旋剑
- 分裂弓
- 火焰杖
- 雷暴锤
- 第 5 层 Boss
- Game Over / Restart

## 里程碑 5：存档、广告结构与交付

- 本地进度保存
- 广告接口预留
- README
- MIT LICENSE
- 多次 commit
- 推送 GitHub

## 开发顺序

1. 先做纯逻辑与测试
2. 再做场景和 UI
3. 再做战斗循环
4. 再做武器表现与 Boss
5. 最后做收尾、验证与推送

## 完成定义

满足以下条件即可视为 MVP 完成交付：

- Godot 项目可启动
- 有开始页和战斗页
- 角色只需左右移动，攻击自动触发
- 怪物刷新、经验升级、三选一、层数推进闭环完整
- 第 5 层有 Boss
- 可 Game Over 并 Restart
- 四把武器可解锁与升级
- 最高层与局外成长保存到本地
- README 和 LICENSE 完整
